PHPue is still early development!

IF you love PHPue Framework, then please help develop better syntax highlighting!

**Every little bit helps!!**

LINUX:
```bash
cp -r PHPue-SyntaxHighlighter ~/.vscode/extensions/phpue.pvue-syntax-0.0.1
```

WINDOWS:
# Command Prompt
```cmd
xcopy PHPue-SyntaxHighlighter %USERPROFILE%\.vscode\extensions\phpue.pvue-syntax-0.0.1 /E /I
```

# Or PowerShell
```powershell
Copy-Item -Path "PHPue-SyntaxHighlighter" -Destination "$env:USERPROFILE\.vscode\extensions\phpue.pvue-syntax-0.0.1" -Recurse
```
---
Move .vscode/settings.json to your project folder!

Run CTRL + SHIFT + P then Reload Window <br>
Do the same and then Change Language Mode: Pvue
